# sudoku

A simple program for solving sudoku puzzles.

The program runs in CLI and takes in a comma delimited file containing a sudoku grid, where '*' represents an empty space.

With the input of:
*,6,1,*,*,*,8,7,*
*,*,*,4,5,*,*,*,3
8,4,*,7,6,*,*,5,*
*,5,*,*,7,*,3,6,*
7,*,*,3,8,5,*,*,1
*,*,4,*,2,*,*,*,*
*,8,*,*,9,6,*,3,*
3,*,*,*,4,7,5,*,*
*,7,2,*,*,*,*,4,*

You should get

5,6,1,9,3,2,8,7,4,
2,9,7,4,5,8,6,1,3,
8,4,3,7,6,1,2,5,9,
9,5,8,1,7,4,3,6,2,
7,2,6,3,8,5,4,9,1,
1,3,4,6,2,9,7,8,5,
4,8,5,2,9,6,1,3,7,
3,1,9,8,4,7,5,2,6,
6,7,2,5,1,3,9,4,8,
